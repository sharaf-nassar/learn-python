# 🐉 Dragon's Tower — Practice Project 🗡️

> **A multi-step adventure project that puts everything you learned in Lessons 1–9 to the test.**

---

## The Story

Long ago, the mighty dragon **Krath** swooped down from the sky and stole the kingdom's golden crown. He flew it to the top of the cursed **Dragon's Tower** and locked it away. Many heroes have tried to climb the tower to bring it home... none have returned.

Today, you've answered the king's call.

By the end of this project, you'll have built a complete adventure game where a hero you create climbs a five-floor tower, fights monsters, finds treasure, and battles the dragon at the top. Will you survive?

---

## How This Project Works

You'll build the game in **4 steps**. Each step adds new features to your game until you have a full adventure!

| Step | What You Build | Save As |
|------|----------------|---------|
| 1 | Hero character creation | `01_hero.py` |
| 2 | Your first battle (gate guardian) | `02_first_battle.py` |
| 3 | Climbing the tower (paths, treasure, traps) | `03_climbing.py` |
| 4 | The full game with the dragon boss | `04_dragons_tower.py` |

**How to work through it:**
1. Open the `.md` file for the step
2. Read the story and the mission
3. Write the Python code yourself in a new `.py` file
4. Test it (the file shows you what it should look like!)
5. Try the bonus challenges
6. When it works, move to the next step

> 💡 **Tip:** You can copy your code from one step to the next and just *add* to it — that's how real game developers grow a game! Or you can start each step fresh if you want extra practice typing.

---

## What You'll Practice

By the time you finish all 4 steps, you'll have used **every single concept** from Lessons 1–9:

- ✅ `print()` and banners (Lesson 1)
- ✅ Variables and `+=` updates (Lesson 2)
- ✅ `input()` and f-strings (Lesson 3)
- ✅ `int()` casting on user input (Lesson 4)
- ✅ `import random` and `random.randint()` (Lesson 5)
- ✅ `if`/`elif`/`else` decisions (Lesson 6)
- ✅ `and`/`or`/`not`, nested `if`s, chained comparisons (Lesson 7)
- ✅ `while` loops and `break` (Lesson 8)
